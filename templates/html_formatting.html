HTML Text Formatting

❮ Previous Next ❯

HTML contains several elements for defining text with a special meaning.

Example

<b>This text is bold</b>
<i>This text is italic</i>
<strong>This is important!</strong>

HTML Formatting Elements

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

HTML <b> and <strong> Elements

The <b> element defines bold text, without extra importance.

The <strong> element defines important text.

<b>Bold text</b>
<strong>Important text</strong>

HTML <i> and <em> Elements

The <i> element defines italic text.

The <em> element defines emphasized text.

<i>Italic text</i>
<em>Emphasized text</em>

HTML <small> Element

The <small> element defines smaller text.

<small>This is smaller text</small>

HTML <mark> Element

The <mark> element highlights text.

<p>Do not forget to buy <mark>milk</mark> today.</p>

HTML <del> Element

The <del> element defines deleted text.

<p>My favorite color is <del>blue</del> red.</p>

HTML <ins> Element

The <ins> element defines inserted text.

<p>My favorite color is <del>blue</del> <ins>red</ins>.</p>

HTML <sub> Element

The <sub> element defines subscript text.

<p>H<sub>2</sub>O</p>

HTML <sup> Element

The <sup> element defines superscript text.

<p>X<sup>2</sup></p>

Extra Tips

  • ✔ Use <strong> instead of <b> for importance
  • ✔ Use <em> instead of <i> for emphasis
  • ✔ Formatting improves readability
  • ✔ Avoid overusing too many styles
