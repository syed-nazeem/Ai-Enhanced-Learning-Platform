HTML Comments

❮ Previous Next ❯

HTML comments are not displayed in the browser, but they can help document your HTML source code.

HTML Comment Tag

You can add comments to your HTML source by using the following syntax:

<!-- This is a comment -->

Note: There is an exclamation point (!) in the start tag, but not in the end tag.

Note: Comments are not displayed by the browser, but they can help document your HTML source code.

Add Comments

With comments you can place notifications and reminders in your HTML code:

<!-- This is a comment -->
<p>This is a paragraph.</p>

<!-- Remember to add more information here -->

Hide Content

Comments can be used to hide content.

<p>This is a paragraph.</p>

<!-- <p>This is another paragraph</p> -->

<p>This is a paragraph too.</p>

You can also hide more than one line:

<!--
<p>This is a paragraph</p>
<p>This is another paragraph</p>
-->

Comments are also great for debugging HTML, because you can comment out lines of code.

Hide Inline Content

Comments can also be used to hide parts in the middle of HTML code.

<p>This is <!-- hidden text --> a paragraph.</p>

Extra Tips

  • ✔ Use comments to explain complex code
  • ✔ Use comments for debugging
  • ✔ Avoid writing sensitive data in comments
  • ✔ Keep comments short and meaningful
