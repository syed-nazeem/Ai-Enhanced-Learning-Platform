HTML Styles - CSS

❮ Previous Next ❯

CSS stands for Cascading Style Sheets. CSS saves a lot of work and controls the layout of multiple web pages at once.

What is CSS?

  • ✔ CSS stands for Cascading Style Sheets
  • ✔ CSS describes how HTML elements are displayed
  • ✔ CSS saves a lot of work
  • ✔ CSS controls layout of multiple web pages

Tip: CSS can be added to HTML in 3 ways: Inline, Internal, External.

Using CSS

CSS can be added to HTML documents in three ways:

  • Inline - using the style attribute
  • Internal - using a <style> element
  • External - using a CSS file

Inline CSS

Inline CSS is used to apply a unique style to a single element.

<h1 style="color:blue;">A Blue Heading</h1>
<p style="color:red;">A red paragraph.</p>

Internal CSS

Internal CSS is defined inside a <style> element in the <head> section.

<head>
<style>
body {
  background-color: lightblue;
}
h1 {
  color: white;
}
p {
  color: black;
}
</style>
</head>

External CSS

External CSS is used to define the style for many pages.

<head>
<link rel="stylesheet" href="styles.css">
</head>

Example of styles.css file:

body {
  background-color: lightblue;
}
h1 {
  color: navy;
}
p {
  color: gray;
}

Best Practice: Always use external CSS for large projects.

CSS Colors, Fonts and Sizes

body {
  background-color: lightblue;
}

h1 {
  color: white;
  font-family: verdana;
  font-size: 300%;
}

p {
  color: black;
  font-family: courier;
  font-size: 160%;
}

CSS Border

p {
  border: 2px solid powderblue;
}

CSS Padding

p {
  padding: 20px;
}

CSS Margin

p {
  margin: 20px;
}

Link to External CSS

Use the <link> tag inside the <head> section:

<link rel="stylesheet" href="styles.css">

You can also link using full URL:

<link rel="stylesheet" href="https://example.com/styles.css">

Extra Tips

  • ✔ Use external CSS for scalability
  • ✔ Keep styles separate from HTML
  • ✔ Use classes instead of inline styles
  • ✔ Maintain consistent design

Chapter Summary

  • ✔ CSS styles HTML elements
  • ✔ Inline CSS is for single elements
  • ✔ Internal CSS is for one page
  • ✔ External CSS is for multiple pages
