HTML Attributes

❮ Previous Next ❯

HTML attributes provide additional information about HTML elements.

HTML Attributes

  • All HTML elements can have attributes
  • Attributes provide additional information
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL.

<a href="https://www.w3schools.com">Visit W3Schools</a>

The src Attribute

The <img> tag is used to embed an image. The src attribute specifies the path.

<img src="img_girl.jpg">

The width and height Attributes

The <img> tag should also contain width and height attributes.

<img src="img_girl.jpg" width="500" height="600">

The alt Attribute

The alt attribute provides alternative text for an image.

<img src="img_girl.jpg" alt="Girl with a jacket">

The style Attribute

The style attribute is used to add styles to an element.

<p style="color:red;">This is a red paragraph.</p>

The lang Attribute

The lang attribute declares the language of the Web page.

<html lang="en">

The title Attribute

The title attribute defines extra information about an element.

<p title="I'm a tooltip">This is a paragraph.</p>

We Suggest: Always Use Lowercase Attributes

HTML is not case sensitive, but lowercase is recommended.

We Suggest: Always Quote Attribute Values

<a href="https://www.w3schools.com">Link</a>

Single or Double Quotes?

Double quotes are most common, but single quotes can also be used.

<p title='John "ShotGun" Nelson'>

Chapter Summary

  • All HTML elements can have attributes
  • Attributes provide additional information
  • Always use lowercase attributes
  • Always quote attribute values
