# AI Enhanced Learning Platform

## 📌 Overview
This project is an AI-powered e-learning platform developed using Flask. It provides interactive tutorials, coding practice, and an AI assistant for real-time doubt solving.

## 🚀 Features
- HTML, CSS, JavaScript tutorials
- Interactive code editor
- Student dashboard
- AI assistant (OpenRouter API)
- Login system UI

## 🛠 Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- AI Integration: OpenRouter API

## 📁 Project Structure
- app.py → Main backend file
- templates/ → HTML pages
- static/ → CSS, JS files

## 🔒 Security
API keys are not exposed in this repository.

## 🎯 Future Improvements
- Database integration
- User authentication system
- Deployment on cloud
