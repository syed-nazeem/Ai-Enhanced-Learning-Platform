HTML Quotation and Citation Elements

❮ Previous Next ❯

In this chapter we will go through the HTML quotation and citation elements.

Example

<p>Here is a quote from WWF's website:</p>

<blockquote cite="http://www.worldwildlife.org">
For 50 years, WWF has been protecting the future of nature.
</blockquote>

HTML <blockquote> for Quotations

The <blockquote> element defines a section that is quoted from another source.

<blockquote cite="https://example.com">
This is a long quotation taken from another source.
</blockquote>

HTML <q> for Short Quotations

The <q> tag defines a short quotation.

<p>WWF's goal is to: 
<q>Build a future where people live in harmony with nature</q>
</p>

HTML <abbr> for Abbreviations

The <abbr> tag defines an abbreviation or acronym.

<p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

HTML <address> for Contact Information

The <address> tag defines contact information for the author.

<address>
Written by John Doe.<br>
Visit us at example.com<br>
Email: test@example.com
</address>

HTML <cite> for Work Title

The <cite> tag defines the title of a creative work.

<p><cite>The Scream</cite> by Edvard Munch</p>

HTML <bdo> for Bi-Directional Override

The <bdo> tag is used to override the current text direction.

<bdo dir="rtl">This text will be written from right to left</bdo>

Extra Tips

  • ✔ Use <blockquote> for long quotes
  • ✔ Use <q> for short inline quotes
  • ✔ Use <abbr> to show full forms on hover
  • ✔ Use <cite> for titles of books, movies, etc.
